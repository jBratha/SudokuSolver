# SudokuSolver

## Description
Main target is to solve [Sudoku](https://en.wikipedia.org/wiki/Sudoku) board.

## Authors  
* [Katorek - Wojtek Jaronski](https://github.com/katorek)
* [Jaro2Gw - Piotr J.](https://github.com/jaro2gw)

### Requirements
* Java
* Maven

### How to run
* clone repo 
* extract to dir
* cd dir
* <del>mvn clean package
* mvn exec: java</del>
